Программа PhotoSorting предназначена для сортировки изображений по годам и месяцам, и сохранения в такой структуре,
в заданой папке.

Алгоритм работы программы:
1. Входные данные:
            Определение папки нахождения файлов с изображениями (source), папки нахождения фотоальбома(destination);
2. Основной блок:
            2.1 Определение файлов, находящихся в папке source;
            2.2 Работа с каждым файлом по следующему алгоритму:
                    2.2.1 Определить, что файл является изображением (имеет расширение jpg);
                    2.2.2 Определить год и месяц съемки фотографии;
                    2.2.3 Скопировать данный файл в папку destination, в подпапку - год (2018) и
                            в подпапку - месяц (июнь). При отсутствии - создать эти папки.
3. Вывод на экран:
            3.1 Для контроля работы программы выводим название файла, год и месяц съемки.
